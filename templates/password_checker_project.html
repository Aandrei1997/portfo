002: Password checker

This project implements a tool that checks if the password we want to use has ever been hacked. The script for this tool can be run from the command line and can accept multiple passwords to verify them.

For this verification of password it is used the API of the following web site:
https://haveibeenpwned.com/
This web site can detect if a password or an email have been ever appear in a leakage of informations.

The way the tool works is the following: it runs the script from the command line with the passwords that will be verified as arguments.

For each password the encryption algorithm SHA1 is applied so as not to send the password to the API in clear format. Also for security reasons, the first 5 characters of the encrypted password are used because the website used uses the key anonymity technique.

The key anonymity technique assumes that the API has a database with all the passwords that have been hacked, and all of these are encrypted with the SHA1 algorithm. So the API will look in this database and it will match all the passwords that start with those 5 characters that the script sends and all of those matches will be sent from API as response. After that all these passwords that are sent by the API are verified by the Python script using the following characters after those 5 initial from the password and verifying if those characters matches with the characters from the passwords returned by the API.

Finally every match is counted and the tool will return the number of appearances of the respective password.
